# v0.3.4.1-public-display-update

这是公开版 v0.3.4 的轻量修复版，不合并私用声息功能。

## 修复内容

- 修复部分 vivo / OriginOS 机型首次打开时只显示左上角一小块的问题。
- Manifest 增加现代 `targetSdkVersion=34`、多屏幕支持和可调整窗口声明，避免系统按旧应用兼容缩放。
- 启动后追加多次布局重测量，首屏会主动重新铺满可用区域。
- 设置页新增「版本与更新」抽屉：显示当前版本、检查更新、更新日志和下载最新版入口。
- 后端新增 `/api/update.json` 与 `/update.json`，可返回更新信息。

## 不包含

- 不包含私用版 v0.3.5 的声息功能。
- 不改变 Render 一键部署、本地部署、Hugging Face 部署、MCP 工具教程。
